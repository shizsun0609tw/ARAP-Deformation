# ARAP_deformation

As rigid as possible deformation
demo:https://youtu.be/UmWazyS02hE

使用方法

1.切換模式
s: selection mode(進入程式時預設)
d: deformation mode

2. selection mode
滑鼠左鍵: 旋轉模型
滑鼠右鍵: 製造handle; 
          "按下->拖拉->放開"這一連串動作所定義的box中包含的所有3D點會成為一個handle

3. deformation mode
滑鼠左鍵: 旋轉模型
滑鼠右鍵: 形變; 被變形的handle是最靠近按下右鍵處的那一個
